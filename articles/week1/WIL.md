이번주 미션을 통해서 백엔드 api개발에 대한 기본과정을 학습했다.

먼저 RESTful api 설계 원칙을 이해했고, 요청과 응답 사이 데이터를 운반하는 DTO를 학습하며 데이터 구조를 설계하는 방법을 알게되었다. 두 번째와 세 번째 미션에서는 실제로 개발을 해보았는데 두 번째 미션에서는 api명세서를 작성해보며 어떤식으로 내가 구현할 api들을 표현할 것인지 정의하고 이걸 따라서 swagger를 활용해 시각적으로 문서화 해보았는데 api명세서도 swagger도 처음 해보는거라서 많은 블로그들을 찾으며 학습했는데 문제해결 능력이 좀 늘은것 같다. 세 번째 미션을 통해서는 쇼핑몰의 재고 검색, 구매, 등록, 추가, 삭제 기능을 담당하는 controller계층을 직접구현하여 보았다. 구현하는데 아는게 없어서 시간이 정말 오래걸렸고 블로그들 ai들의 도움을 조금 받았지만 도움 받은 후에는 코드들을 다시 이해하려 노력하고 정리했다. 앞으로들의 미션에서는 조금 더 내 힘을 더해 구현할 수 있을 것 같다.
